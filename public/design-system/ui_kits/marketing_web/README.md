# Sembada.id — Marketing Site UI Kit

Single-page marketing landing page. Pure HTML/CSS for max editability.

## Files
- `index.html` — full landing: nav, hero, logo cloud, features, quote, CTA strip, footer.

## Structural notes
- Hero h1 uses Fraunces 72px with italicized accent on "tumbuh".
- Hero visual is a tilted sogan card with batik kawung overlay, holding a floating white phone-card mock.
- Quote section is full-bleed sogan with parang pattern overlay.
- Footer is tanah-900 (deep brown) — only place we go that dark.
- Every Fraunces use sets `font-variation-settings: "opsz" XX, "SOFT" 30` for the rounded, warm cut.
