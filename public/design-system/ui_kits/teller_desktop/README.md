# Sembada Teller — Desktop UI Kit

Single-screen kasir/teller terminal. Optimized for keyboard-first input (F-keys + Enter).

## Files
- `index.html` — full teller terminal layout: dark sidebar (tanah-900), member lookup panel, transaction form, recent transactions log.

## Notes
- Density is higher than other surfaces — this is for staff who do this all day.
- Sidebar is the deepest brown (tanah-900) so it disappears in operator focus; main panels are cream surface.
- F-key hints in the toolbar communicate keyboard-first usage.
- Amount input uses Fraunces display so big numbers read confidently.
